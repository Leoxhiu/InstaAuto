For Windows

1. Download python from here, https://www.python.org/downloads/

2. Move your file to Desktop for easier configuration

3. Open .env with notepad

4. Adjust configuration based on your preference, replace
USERNAME=(your instagram account username)
PASSWORD=(your instagram account password)
TARGET_POST=(url,link of post/reel to be shared)
USER_PER_BATCH=(number of users to share post at one time)

5. Open CMD

6. Type 
-> cd Desktop/InstaAuto

7. Type
-> ./env/Scripts/activate

8. Type
-> pip install -r .\requirements.txt

9. Type
-> python main.py
